# Podmanインストール
1. 事前にhomebrewのインストールが必要
    ```bash
    % /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
1. podman-desktopのインストール
    ```bash
    % brew install podman-desktop
    ```

# 環境
* podman version 5.3.1
* MacOS 15.2
